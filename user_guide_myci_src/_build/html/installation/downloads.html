GitHub¶

Git es un sistema de control de versiones distribuido.

El acceso público Git está disponible en GitHub. Tenga en cuenta que si bien no se escatiman esfuerzos para mantener esta base de código funcional, no se puede dar garantía alguna del código tomado de la rama develop del proyecto.
